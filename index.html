HDHomeRun HLS Streaming

Intro

This project is a web frontend to interface with the HDHomeRun products and FFmpeg in order to provide a mobile ready VBR HLS live stream of a channel.

Chromecast

We now support Chromecast streaming using desktop Chrome browsers. Start the stream in a Chrome browser with the cast extension installed and the cast icon will appear when Chromecast is available.

Requirements

  • Tested on Mac OS X and Ubuntu
  • PHP enabled webserver
  • FFmpeg with libx264 support
  • hdhomerun_config

Installation

  • Clone this repo: git clone https://github.com/Logan2211/hdhomerun_hls.git
  • In html/callback.php configure the path to hdhrstream.class.php
  • Copy html/* to a PHP enabled directory on your webserver. Make sure that your web server has write permissions to this directory so ffmpeg can write the stream segments here.
  • Configure the settings at the top of hdhrstream.class.php. The main options involved in the initial configuration are listed below.
    • target_ip should be set to the IP of the system hosting this app
    • ffmpeg_base must point to your FFmpeg binary
    • stream => path must be set to the html directory where index.html, callback.php, etc reside. This is the directory where VLC will output the m3u8 HLS playlists and transcoded video segments.
    • num_tuners will typically be 3 on HDHomeRun Prime, may be different on other HDHR models.

Advanced/Optional Settings

  • In html/index.html there are several channel_filter_* settings at the top of the file which enable display filtering of channels. It is disabled by default but I use this to display only HD channels.

Usage

Once configured navigate to the index.html URL on your device and attempt to start the stream. Check the web server error log and transcoder log file (configured in hdhrstream.class.php) if the stream does not start.

HLS capable players are iPhone, iPad, Safari on Mac, Chromecast, VLC, etc. In the case of browsers that do not natively support HLS streaming we serve a Flash fallback HLS player.
